# WordMD spec folder

This folder is the canonical home for project planning and operating docs.

- `SPEC.md`: product scope, technical shape, and auth_family declaration.
- `DEPLOY.md`: build, release, installer, and rollback runbook.
- `punchlist.md`: current ideas, bugs, and follow-up work.
- `WORK_LOG.md`: reverse-chronological project work log.
- `card.json`: ReboundMan card metadata.
- `FEEDBACK.md`: feedback intake and loop policy.
